# Generic Sub-Agents

이 디렉터리는 템플릿 저장소에서 재사용할 수 있는 일반 목적 Codex 서브에이전트 역할 정의를 둔다.

## Defined Roles

- `architecture`
- `runtime`
- `gitops`
- `quality`
- `api`
- `orchestrator`
- `ci`
- `specs`
- `ui`
- `security`

## Notes

- 역할 이름과 설명은 특정 서비스명에 묶지 않는다.
- 제품/회사/실환경 고유 정보는 이 디렉터리에 저장하지 않는다.
- 역할 설명은 템플릿 수준의 일반 책임만 다룬다.
