# OTRO Alias

이 파일은 root alias다. canonical OTRO skill body와 scripts/references/schemas는 [`otro/SKILL.md`](./otro/SKILL.md)를 기준으로 사용한다.

## Rule

- `sdd-development`, `ralph-loop`, `planning-with-files`는 별도 skill surface로 유지한다.
